# Compass — AI Medical Emergency Intelligence for California

> *"Always pointing you the right way"*

Compass is an AI system that protects medically vulnerable Californians — people on home oxygen, dialysis, ventilators, and power wheelchairs — by predicting energy and disaster risks before they become emergencies, and delivering personalised, condition-specific action plans to patients and their caregivers.

Built for the **2025 AI Hackathon** — Theme: *Design and build AI systems that meaningfully improve how people live, work, and interact with the world.*

---

## The Problem

**200,000+ Californians** depend on electricity to stay alive. Home oxygen concentrators, dialysis machines, ventilators, powered wheelchairs. When the grid fails, they are the most vulnerable — and the least prepared.

The current system fails them in three ways:

| Gap | What happens today | What Compass does |
|---|---|---|
| Awareness | Generic robocall hours before an outage | AI predicts risk 48–72 hrs ahead, personalised by ZIP + condition |
| Preparation | Same generic checklist for everyone | Condition-specific action plan generated by Gemini for each user |
| Response | No welfare check during an event | AI voice agent calls, confirms safety, escalates to caregiver |

## RocketRide Integration

RocketRide is the AI pipeline orchestration layer. It runs inside VS Code and manages the two core AI workflows:

### Pipeline 1 — Data Ingestion Pipeline (`pipeline/ingestion_pipeline.json`)
Runs every 15 minutes via Cloud Scheduler trigger.

```
[CAISO API] ──┐
[NOAA API]  ──┤──► [Normalise Node] ──► [Risk Score Node] ──► [BigQuery Write Node]
[CAL FIRE]  ──┤
[FEMA]      ──┘
```

### Pipeline 2 — Advisory Generation Pipeline (`pipeline/advisory_pipeline.json`)
Triggered when a user's risk score crosses threshold (>60).

```
[Risk Score Input] ──► [User Profile Fetch] ──► [Gemini Pro Node] ──► [Action Plan Output]
                                                        │
                                              [Equipment DB Lookup]
```

### Running RocketRide Locally

```bash
# Install VS Code extension from marketplace
# Extension ID: RocketRide.rocketride

# Install SDK
pip install rocketride

# Run ingestion pipeline
python -c "
import asyncio
from rocketride import RocketRideClient

async def run():
    async with RocketRideClient(uri='http://localhost:5565') as client:
        result = await client.use(filepath='pipeline/ingestion_pipeline.json')
        token = result['token']
        response = await client.send(token, 'run_ingestion')
        print(response)

asyncio.run(run())
"
```

## Three Feature Pillars

### 1. Predictive — AI Risk Scoring Engine
- Pulls CAISO grid stress, NWS weather, CAL FIRE PSPS zones, FEMA flood maps
- Vertex AI time-series model forecasts risk 48–72 hours ahead
- Weighted composite score (0–100) personalised by ZIP + user profile
- Updates every 15 minutes via RocketRide ingestion pipeline

### 2. Preventive — Onboarding + Action Planning
- Onboarding captures: condition, equipment, backup power, caregiver, mobility
- Gemini Pro generates condition-specific ranked action checklist
- Readiness score updated continuously between events
- Monthly reminders to confirm backup supplies and contacts

### 3. Reactive — AI Welfare Check Agent
- Push notification when score crosses 60
- SMS backup if no app response within 2 minutes
- AI voice call if no SMS response — conversational welfare check
- Immediate caregiver alert with live status dashboard
- Case worker dashboard flags highest-risk clients for direct outreach

## Regulatory Tailwind

- **California Medical Baseline Program** — 200,000+ enrollees across PG&E, SCE, SDG&E
- **CPUC PSPS Citation Program (2023)** — utilities fined for inadequate notification to Medical Baseline customers
- **California SB 1370** — enhanced outreach legally required for medically vulnerable utility customers
- **AB 1771 (2023)** — statewide extreme heat preparedness plan mandated
- **California Gov Code §8593.3** — AFN (Access and Functional Needs) integration required in all emergency planning

Compass is the AI layer that makes these legal mandates actually work.
